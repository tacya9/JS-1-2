### 2.10.25 Человек вводит время, чтобы поставить будильник. Выводится сообщение для ввода часов: "Введите часы" Потом для ввода минут: "Введите минуты" Есть условия у этого будильника: - значения часов может быть 0 - 23 - значения минут может быть 0 - 59 - будильник можно ставить только утром с 5:00 до 8:00 включительно Если часы введены неверно, тогда сразу появляется алерт "Неправильные данные" и сообщение для минут не появится. И если все условия выполняются, тогда вывести алерт "Будильник готов", иначе "Неправильные данные"
